# Accio 对标界面设计方案 V2

**版本**: 2.0  
**日期**: 2026-02-08  
**作者**: Manus AI

---

## 1. 整体布局

采用两栏式布局：
- **左侧**: 垂直导航栏
- **中间**: 主区域（搜索框 + 快捷按钮）

## 2. 左侧导航栏

- **背景色**: 浅灰色 (`#f7f7f8`)
- **内容**:
  - Logo + 折叠按钮
  - 首页
  - 历史记录
  - 我的列表
  - 消息
  - 邀请有礼
  - 下载 APP
  - 联系我们

## 3. 中间主区域

- **背景色**: 白色
- **顶部栏**: 语言/货币选择器 + 积分 + 免费试用 + 用户头像
- **标题**: "Accio" + "AI智能采购，一问搞定"
- **大搜索框**: 圆角，带阴影，占位符为"描述您的需求..."
- **快捷按钮**: 圆角胶囊式按钮

## 4. 视觉风格

- **浅色主题**: 白色和浅灰色为主
- **简洁的图标**: 使用 Lucide React 图标库
- **圆角**: 大量使用圆角，营造亲和感
- **阴影**: 使用柔和的阴影，增加层次感

## 5. 实施计划

1. **创建主布局**: `web/src/app/accio/layout.tsx`
2. **实现左侧导航栏**: `web/src/components/accio/sidebar.tsx`
3. **实现主区域**: `web/src/app/accio/page.tsx`
4. **更新路由**: 将 `/` 重定向到 `/accio`
