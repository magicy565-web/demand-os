# Accio 对标设计方案

**版本**: 1.0  
**日期**: 2026-02-08  
**作者**: Manus AI

---

## 1. 核心设计理念

### "一句话完成复杂任务"

抛弃传统的聊天界面，采用**搜索式界面**，用户只需输入一个需求，系统就会自动规划、执行并呈现结果。

---

## 2. 界面设计

### 2.1 整体布局

- **顶部**: 大而醒目的搜索框
- **中部**: 执行计划卡片
- **底部**: 结果展示区域

### 2.2 搜索框

- **样式**: 简洁的圆角矩形，有阴影效果
- **交互**: 输入时高亮，支持语音输入

### 2.3 执行计划卡片

- **样式**: 卡片式设计，每个步骤一行
- **内容**: 图标 + 标题 + 状态指示器
- **交互**: 当前步骤高亮，可展开查看日志

### 2.4 结果展示

- **卡片式布局**: 每个结果一个卡片
- **多种形式**: 支持文本、图表、表格
- **可交互**: 支持筛选、排序、导出

---

## 3. 视觉风格

- **深色模式**: 主背景色为深色（如 `#0a0a0f`）
- **霓虹绿色强调**: 使用醒目的绿色（如 `#00FF00`）作为主强调色
- **现代简洁**: 去除不必要的装饰，突出内容

---

1. **创建搜索式界面**: `web/src/app/search/page.tsx`
2. **实现执行计划卡片**: `web/src/components/search/execution-plan-card.tsx`
3. **实现结果渲染器**: `web/src/components/search/result-renderer.tsx`
4. **更新后端 API**: 支持搜索式交互
