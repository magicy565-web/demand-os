### 12. **坐标转换逻辑错误**
**当前问题**：所有标注点都在右侧堆叠。

**修复方案**：
```typescript
// 正确的经纬度到 SVG 坐标转换
function latLngToSVG(lat: number, lng: number, svgWidth: number, svgHeight: number) {
  // 中国地图的经纬度范围
  const minLng = 73.5;  // 最西
  const maxLng = 135.0; // 最东
  const minLat = 18.0;  // 最南
  const maxLat = 53.5;  // 最北
  
  // 墨卡托投影
  const x = ((lng - minLng) / (maxLng - minLng)) * svgWidth;
  const y = ((maxLat - lat) / (maxLat - minLat)) * svgHeight;
  
  return { x, y };
}
```
