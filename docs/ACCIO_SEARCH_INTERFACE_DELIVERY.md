## 🎨 核心设计特点

### 1. 深色模式

- **背景色**: `#0a0a0f` (深黑色)
- **卡片背景**: `slate-900` (深灰色)
- **文字颜色**: 白色和灰色

### 2. 霓虹绿色强调

- **主强调色**: `emerald-400` (翡翠绿)
- **辅助强调色**: `cyan-400` (青色)
- **渐变按钮**: `from-emerald-500 to-cyan-500`

### 3. 大搜索框

- **高度**: 64px (h-16)
- **圆角**: 16px (rounded-2xl)
- **边框**: 2px 实线，焦点时变为翡翠绿
- **图标**: 左侧搜索图标，右侧搜索按钮

### 4. 执行计划卡片

- **状态着色**:
  - 完成: `emerald-950/20` 背景 + `emerald-900` 边框
  - 进行中: `cyan-950/20` 背景 + `cyan-900` 边框 + 阴影
  - 失败: `red-950/20` 背景 + `red-900` 边框
  - 等待: `slate-800/50` 背景 + `slate-700` 边框

- **可展开**: 点击步骤卡片可展开查看日志和结果
- **连接线**: 步骤之间有连接线显示顺序

### 5. 结果展示

- **卡片式布局**: 最终结果以卡片形式展示
- **JSON 格式**: 结果以格式化的 JSON 展示（后续可优化为表格或图表）

---

## 🚀 使用方式

### 1. 访问页面
```
http://localhost:3000/search
```

### 2. 输入需求
在搜索框中输入需求，例如：
```
我想开发一款智能蓝牙音箱
```

### 3. 查看执行计划
系统会自动生成执行计划，并实时展示每个步骤的进度。

### 4. 查看结果
任务完成后，最终结果会显示在底部的"最终结果"卡片中。
