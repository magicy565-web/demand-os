## 问题诊断

从截图看，标注位置似乎比计算的坐标更加集中。可能的原因：

1. **SVG viewBox 与实际地图不匹配** - SimpleMaps 的 SVG 可能有自己的坐标系统
2. **需要添加偏移量校准** - 可能需要根据实际地图调整基准点
3. **地图投影方式** - SimpleMaps 可能使用了特定的投影方式，不是简单的线性映射

## 解决方案

### 方案 A：手动校准（推荐）

通过参考点校准坐标转换：

1. 选择2-3个已知城市（如北京、上海、广州）
2. 在 SVG 中找到它们的实际坐标
3. 建立校准映射关系
4. 调整转换公式

### 方案 B：使用 SVG 原生坐标

SimpleMaps 的 SVG 中每个省份都有自己的 path，可以：

1. 提取每个省份的中心点坐标
2. 直接使用 SVG 坐标系统
3. 不需要经纬度转换

### 方案 C：调整地图显示区域

如果坐标转换正确但显示不准确，可以：

1. 调整 SVG viewBox 聚焦到东南沿海
2. 放大产业带密集区域
3. 提升标注的可见度和间距
